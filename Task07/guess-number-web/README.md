Угадай число.
Веб-версия классической игры "Угадай число"

Описание игры
Компьютер загадывает число в заданном диапазоне, а игрок должен угадать его за ограниченное количество попыток. После каждой попытки игрок получает подсказку: "больше" или "меньше".

Онлайн версия
Игра доступна онлайн: https://zerollka.github.io/guess-number-web
